# Head Ball

Een één-tegen-één voetbalspel met grote hoofden en snelle actie.

## Built with AI

Dit spel is volledig gebouwd met **GitHub Copilot Coding Agent** en het **GPT-4.1** model.

### Development History

- [PR #XXX](https://github.com/rajbos/game-center/pull/XXX) - Initiële implementatie van Head Ball

## How to Play

- Gebruik de pijltjestoetsen om te bewegen
- Spatie om te springen/koppen
- Scoor doelpunten door de bal in het doel van de tegenstander te krijgen

## Mobile Support

- Responsief ontwerp voor telefoons en tablets (minimaal 375px breed)
- Touch controls worden toegevoegd in een latere versie
- Spelinstructies zijn zichtbaar voor desktop en mobiel
- Getest op mobiele viewport
